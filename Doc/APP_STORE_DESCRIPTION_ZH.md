# App Store Connect - 中文内容

## 应用名称
时刻+

## 副标题
记住你的每个瞬间

## 推广文本
记住照片里的时刻。在地图上重温那些时间和地点。

## 描述
照片越来越多，却越来越难找。时刻+ 换个角度看它们——不是一张张图片，而是一段段记忆。

• **按时间和地点归类** - 同一次旅行、同一个下午、同一家咖啡店的照片，本来就该在一起。

• **记忆地图** - 看看你去过哪里。点开一座城市，放大到某个街区，重新发现那些快要忘记的地方。

• **添加笔记** - 有时候照片需要一点背景。记下当时在想什么，和谁在一起。

• **留在手机里** - 不上传云端。你的照片只属于你。
